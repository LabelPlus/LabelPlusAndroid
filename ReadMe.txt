Choose one file of the project dir when you new/open a project.
After finish new/open, press 'next' to start your work.
Work will be saved while screen changes.

Use single finger to add label, use two finger to drag and zoom.

新建、加载项目时，任选目录下的一个文件即可。
新建、加载项目后按下一张可开始工作。
工作成果会在切换屏幕时自动保存在图片目录下的 Result.txt 文件里。

单指点击添加标签，双指进行移动、缩放。
